# CodeArena 🏆

A LeetCode clone where users can solve coding problems 
and get instant feedback on their solutions.

## What I Built

- User authentication with GitHub OAuth (Clerk)
- Problem creation and management (Admin only)
- Online code editor with multiple language support
- Real-time code execution using Judge0
- Submission history tracking
- Custom problem playlists
- User profile with stats
- Dark/Light mode

## Tech Stack

- Next.js 15
- Prisma + PostgreSQL
- Clerk Authentication
- Judge0 (Self-hosted code execution)
- shadcn/ui + Tailwind CSS
- Docker
